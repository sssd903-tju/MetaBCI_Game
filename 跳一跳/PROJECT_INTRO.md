# 跳一跳 MI-BCI 项目介绍与创新点

## 项目概述

**基于前额叶 EEG 头环的运动想象脑机接口游戏系统**——让玩家通过左右手运动想象（Motor Imagery, MI）控制角色跳跃方向，无需任何肢体动作。

```
┌─────────┐  Serial   ┌─────────────┐   LSL    ┌──────────────┐  WebSocket  ┌──────────┐
│ EEG 头环 │ ────────→ │ 串口→LSL 桥接 │ ───────→ │ 在线 MI 分类器 │ ───────────→ │ Godot 游戏│
│ (Fp1/Fp2)│ 921600   │              │  250Hz   │ (FAA+RF/SVM) │  :8767      │ (跳一跳)  │
└─────────┘           └─────────────┘          └──────────────┘             └──────────┘
```

**核心体验：** 想象左手运动 → 角色向左跳 | 想象右手运动 → 角色向右跳 | 10 层关卡竞速

---

## 一、系统架构

### 硬件层
- **脑电设备**：8 通道 Brain Cube 放大器，实际仅使用 Fp1/Fp2（前额叶）
- **通信协议**：串口 921600 baud，44 字节帧（4ch × 3B INT24），1000Hz 采样
- **头环佩戴**：Fp1/Fp2 位于前额，阻抗 < 5kΩ

### 软件栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 数据采集 | Python + pyserial | 串口读帧 → 解析 24-bit 有符号 EEG |
| 数据标准化 | **LSL** (Lab Streaming Layer) | 实时流式传输，跨语言/跨进程，250Hz 4ch |
| 信号处理 | SciPy | 0.5-45Hz 带通 + 50Hz 陷波 + z-score 归一化 |
| 特征提取 | 自研 **FAA** (前额叶 Alpha 不对称) | 6 频带 × 4 + 2 频带 × 3 ERD = 30 维特征 |
| 分类器 | **Random Forest / SVM-RBF** | sklearn，离线训练 LOOCV 验证 |
| 在线推理 | Python asyncio + WebSocket | LSL 拉流 → 环形缓冲 → 试次触发分类 |
| 游戏引擎 | Godot 4 (GDScript) | 2D 物理跳跃，关卡模式 + 离线训练模式 |
| 数据记录 | JSONL | 试次级时间戳 + 标签 + BDF 脑电文件 |

### 数据流

```
EEG硬件 → Serial(AA+12B+BB) → serial_lsl_bridge.py → LSL "brain-cube-eeg"
    → fp1fp2_online.py (环形缓冲, trial_start触发)
    → 带通0.5-45Hz → 陷波50Hz → z-score
    → 30维FAA特征 → RF/SVM分类 → left/right + 置信度
    → WebSocket :8767 → Godot Game.gd → 关卡跳跃
```

## 二、核心创新点

### 创新 1：仅用 Fp1/Fp2 前额叶实现 MI 分类（硬件简化）
**传统方案：** 运动想象 BCI 几乎全部使用 C3/C4/Cz 等感觉运动区电极，需要佩戴 8-64 通道脑电帽，准备时间长、成本高、用户体验差。

**本方案：** 仅使用前额叶 Fp1/Fp2 双通道（消费级头环即可覆盖），通过 **前额叶 Alpha 不对称（FAA）** 实现左右手 MI 分类。

**科学依据：** 前额叶在运动规划、注意力和认知控制中发挥关键作用。左手 MI → 右侧前额叶激活 → FP2 alpha 功率下降；右手 MI → 左侧前额叶激活 → FP1 alpha 功率下降。通过 (FP2-FP1)/(FP2+FP1) 的不对称指数可有效区分。

**成果：** 40 试次训练，RF 分类器 LOOCV 准确率 **77.5%**，SVM-RBF 达 **80%**。

### 创新 2：30 维多频带 FAA 特征工程（信号处理创新）
**传统方案：** 仅用 alpha 频段 (8-13Hz) 的 ERD/ERS 或单一功率比。

**本方案：** 设计 **30 维复合特征**，覆盖 6 个频带的多维度信息：

| 特征组 | 维度 | 说明 |
|--------|------|------|
| 多频带绝对功率 | 6×4=24 | θ/α/low-β/high-β/β/broadband 每频带 log_FP1, log_FP2, FAA, log_total |
| ERD 特征 | 2×3=6 | α/β 频带 ERD_FP1, ERD_FP2, lateral_ERD |

**关键洞察：**
- FAA（不对称指数）天生尺度不变，对设备差异鲁棒
- 多频带覆盖捕捉不同认知成分（θ=注意力，α=抑制，β=运动准备）
- log 变换稳定方差，适合线性/非线性分类器

### 创新 3：z-score 窗口归一化实现跨设备迁移（工程创新）
**问题：** 训练数据（BDF 文件，电压量纲 ~0.5V）与实时数据（Brain Cube 串口，ADC 原始值 ~866k）尺度相差 6 个数量级，直接推理完全失效。

**方案：** 在特征提取前，对每个试次的 Fp1/Fp2 窗口独立做 z-score 归一化（`x / std(window)`），消除全局尺度差异，仅保留频带间的相对关系。

**效果：** 同一模型在 BDF 回放和实时设备上均可工作，无需重新训练。

### 创新 4：试次驱动的游戏-脑电同步协议（系统设计创新）
**传统方案：** BCI 游戏通常采用连续滑动窗口分类，存在标签与游戏事件不对齐的问题。

**本方案：** 设计 **trial_start 事件驱动的同步协议**：

```
游戏 Phase:  START(0-2s) → MI_TASK(2-6s) → JUMP(6-7s) → SCORE(7-8s) → RELAX(8-10s)
             ↓trial_start                      ↑读mi_decision   ↓落地清除
EEG:         ├─ baseline(2s) ──┼── task(2s) ──┤
分类器:                       ← 4s 收集 → classify → left/right
```

**优势：**
- 每个试次精确对齐 baseline 和 task 窗口
- 自动记录 ground truth（正确跳跃方向），支持离线训练
- 同层去重 + 落地清除，避免旧标签污染新试次
- JSONL 格式完整记录每次试次的脑电时间戳

### 创新 5：离线/在线一体化训练闭环（工程创新）
**传统方案：** 数据采集和模型训练分离，需要多次迭代才能优化。

**本方案：** 游戏内建 **关卡训练模式（VERTICAL_TRAIN_OFFLINE）**，实现采集→训练→推理的闭环：

1. **采集阶段**：玩家在游戏中完成 40 次 MI 试次，自动记录 EEG(BDF) + 标签(JSONL)
2. **训练阶段**：一键运行 `train_fp1fp2.py`，对比 LDA/SVM/RF 三种分类器
3. **推理阶段**：最佳模型直接部署到在线服务器，实时驱动游戏

**GUI 集成**：`mi_bci_gui.py` 提供完整的图形界面——串口桥接、信号监视、BDF录制、模型训练、在线推理，一站式操作。

### 创新 6：多层冗余的鲁棒性设计
- **服务器端**：同层 trial_start 去重 + 8s 超时兜底 + 环形缓冲保证数据完整性
- **游戏端**：序号检查（防乱序）+ TTL 检测（1500ms 过期）+ 连续确认去抖
- **特征端**：NaN/Inf 保护 + z-score 防止除零
- **通信端**：WebSocket 自动重连（0.75s 间隔）+ LSL 流断线恢复

## 三、量化性能

### 分类准确率（LOO 交叉验证，40 试次，Fp1/Fp2）

| 分类器 | 特征维度 | 准确率 | 说明 |
|--------|---------|--------|------|
| FAA 符号 | 1 | 55.0% | 简单阈值 |
| LDA | 30 | 60.0% | 线性边界 |
| **SVM-RBF** | 30 | **80.0%** | 非线性核 |
| **Random Forest** | 30 | **77.5%** | 集成学习 |

### 系统延迟

| 环节 | 延迟 |
|------|------|
| 串口→LSL | < 5ms |
| 信号处理 | < 10ms |
| 特征提取+分类 | < 5ms |
| WebSocket 传输 | < 1ms |
| **端到端（trial_start→结果）** | **~4s**（有意设计的窗口长度） |

### 统计显著性

- Fp1/Fp2 Alpha 不对称 left vs right：**p = 0.0010**（高度显著）
- FAA_task: Left MI = 0.709±0.126, Right MI = 0.835±0.081

## 五、与传统方案对比

| 维度 | 传统 MI-BCI | 本方案 |
|------|------------|--------|
| 电极数量 | 8-64 通道 | **2 通道 (Fp1/Fp2)** |
| 电极位置 | 感觉运动皮层 (C3/C4) | **前额叶（头环可覆盖）** |
| 佩戴时间 | 10-30 分钟（导电膏） | **< 1 分钟（干电极）** |
| 分类特征 | ERD/ERS, CSP | **多频带 FAA + ERD（30 维）** |
| 分类器 | 通常仅 LDA | **LDA/SVM/RF 对比择优** |
| 跨设备 | 需重新标定 | **z-score 归一化自适应** |
| 数据采集 | 分离式（采集→离线分析） | **游戏内闭环（采集→训练→推理）** |
| 用户体验 | 实验室环境 | **消费级游戏体验** |
